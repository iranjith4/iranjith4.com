Move It

It's just a classic puzzle game. The main objective is, you need to assemble the image pieces that are scrambled. Initially you need to select an image from the set of given images, or you can select your own image from Photos, or even you can snap your own photo. The difficulty levels will start from 3x3 Pieces and max is 6x6. Slide and Join to the Original Pieces.

Technologies : Xcode (IDE), Objective-C, Sprite Kit, Appsaholic SDK, GameKit, GameCenter, Push Notifications, Flurry Analytics, Crashlytics

Tap It

Tap It is completly Native. You will be having 3 shapes running in the screen from left to right, either only the outline, or the Filled one, with any one of the 3 colors (Red, Green, Blue). At the Centre, You will be having 3 Coloured bubble, each carrying one shape. The Objective of the game is, If the running shape is filled one, the tap on the Bubble corresponding to the color of the Shape. If the running shape is filled one, tap on the Bubble corresponding to the shape. Still Confusing ? Have a try.

Technologies : Xcode (IDE), Objective-C, Core Animation, UIAnimation, Appsaholic SDK, GameKit, GameCenter, Push Notifications, Flurry Analytics, Crashlytics
